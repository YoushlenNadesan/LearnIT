CHAPTER 3

In this chapter you will learn what Arithmetic and Comparison operators are and how they are implemented.

 

Python Operators

Operators are special symbols in python that carry out arithmetic or logical computation. Python divides operators into different groups,we will focus on 3 python operators:

1. Arithmetic operators
2. Assignment operators
3. Comparison operators

 

Arithmetic Operators

Arithmetic operators are used with numeric values to perform common mathematical operations. Here is a table of arithmetic operators:

Operator name

Operator

Explanation

Example

Addition

+

Add two operands

3 + 4

Subtraction

-

Subtract second operand from the first

5 - 2

Multiplication

*

Multiplies both operands

8 * 8

Division

/

Divides numerator by denominator

20 / 10

Modulus

%

Computes the remainder of dividing first operand by second operand

61 % 10

Exponential

**

Performs exponential (power) calculation on operands

2 ** 2

Floor division

//

The division operand where the results is the quotient in which digits after the decimal point are removed ,but if one of the operands is negative the result is rounded away from zero

50 // 10

 

Comparison Operators

Comparison operators are used to compare values .It returns either true or false according to their condition. Here is a list of comparison operators:

Operator name

Operator

Explanation

Example

Greater Than

> 

True if left operand is greater than right operand

5 > 2

Less Than

< 

True if left operand is less than the right operand

12 < 32

Equal To

==

True if both operands are equal

44 == 44

Greater Than or Equal To

>=

True if left operand is greater than or equal to the right operand

12 >= 42

Less Than or Equal To

<=

True if left operand is less than or equal to the right operand

67 <= 3

Not Equal To

!=

True if left operand is not equal to right operand

2 != 3

 

Assignment Operators

Assignment operators are used to assign values to variables. These operators can be numeric, date, system, time, and text. Here is a table of assignment operators:

Operator

Explanation

Example

+ =  

This operator adds the right operand to the left operand and stores the results in the left operand

X += 5

- = 

This operator subtract the right operand from the left operand stores the result in the left operand

X -= 5

* =  

This operator multiplies the right operand with the left operand  storing the results in the right operand

X *= 5

/=

This operator divides the left operand by the right operand ,storing the results in the left operand

X /= 5

%=

This operator takes the module of the left operand ,using the right operand and stores the results in the left operand

X %= 5

**=

This operator raises the exponential power of the left operand to the value of the right operand  and stores the results in the left operand

X **= 5
