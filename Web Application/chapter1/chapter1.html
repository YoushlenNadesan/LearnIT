CHAPTER 1

In this chapter we will cover variables, data types and input/output.

 

Variables

Variables are basically containers for storing data values. These variables can be anything! They can consist of 1 or more alpha-numeric (A-z and 0-9) and or underscores. Although variables cannot start with a digit.

Variables are case sensitive, for example the variable NUM is not the same as num and is not the same as Num and so on for each variation of num.

A variable will be attached to data with an equals sign (=).

Example:

Num = 24
myName = “Andre”

 

Data Types

A data type essentially is what gets stored into your variable. Variables are containers for storing data value.

There are 3 fundamental data types.

Integer (int) – an integer is any whole number that can be both positive and negative.
Float (float) – a float will be numbers that have remainders such as 5.5, 4.67, 3.123…etc.
String (str) – a string is usually used to represent text but can also represent numbers. When attaching a string to a variable it must be place in inverted commas like shown in this example: myName = “Andre”

Input/Output

Input allows a user, like you for example, to “input” details. These details will be then processed by the computer and it will then provide you with an output by displaying it for you to see.

In order to prompt a user to input information we need to use the input() function.

To output the result we need to use the print() function.

Example:

myName = input(“What is your name? “)
print(“Hello ” + myName)

Output
What is your name? Andre *your input in italic*
Hello Andre

*notice in the print function when you type the string hello in inverted commas it prints exactly what is typed whereas myName has no inverted commas because it is stated as a variable. If Hello had no inverted commas it will give an error because the computer will think that Hello is variable but it is not stated.*
