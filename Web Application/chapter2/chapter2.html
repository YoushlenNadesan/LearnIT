CHAPTER 2

In this chapter you will learn how to manipulate stings and extract information out of them.

 

String Functions

Since strings usually consist of text you can manipulate how that text is displayed, for example you ca can make any letter in the string a capital letter or you can make it a lower case letter.

You can also extract information such as the string length or how many times a specific character appears within the string.

Here is a table of all the functions you can use for strings.

Explanation

Example
