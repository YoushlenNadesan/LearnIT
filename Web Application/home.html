Welcome to L.iT

 

Hello world

One of the bigger parts of IT is coding/programming but it is notoriously known for being complicated and difficult to understand and master. Well we here at L.iT have designed this web based application to teach you the basic concepts of coding in such a manner that you will find easy to understand.

L.iT is for everyone! It doesn’t matter whether you are a pre-teen, teenager, an adult or part of the older crew, you will be able to learn how to code.

L.iT will teach you how to code by giving you short explanations and examples to read through. After each chapter you will be given a short exam so that you can see how far you’ve come!

L.iT is your small stepping stone into the world of programming. You will be learning to code in a programming language called Python. There are many different programming languages but it can be time consuming to teach you all these languages! Don’t worry though, the concepts are transferrable.
